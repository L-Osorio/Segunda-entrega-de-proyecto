Nombre proyecto : Inventario para un tienda
Grupo #11
1.Laura Alejandra Osorio
2. Sofia Quimbay Cadena

Descripción del proyecto : Programa que permite llevar a cabo un inventario para tienda, donde se puede guardar nombre,
precio, fecha de vencimiento y cantidad de producto para facilitar el manejo de información al dueño de la tienda
